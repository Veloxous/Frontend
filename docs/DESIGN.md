# Veloxous Design Documentation

## Overview
This document contains the links and references to the high-fidelity designs and design system for Veloxous. 

## Figma Design Links
- **Desktop & Mobile High-Fidelity Screens**: [Veloxous Figma Design](https://www.figma.com/design/8pA4kjAi1OvO1fVhdy1G0b/veloxous?timeline=keyframe&node-id=0-1&p=f&t=Oz2v5wuG53Fz2VeH-0)

### Contents of Design File
1. **Design System**: Contains the color palette, typography (Space Grotesk & Inter), components, and style guides.
2. **Landing Page**: Value proposition, product overview, how escrow works, verified repair network.
3. **User Dashboard**: Wallet balance, Trust Score, transaction history, and active escrows.
4. **Marketplace Discovery**: Product details, device listing wizard, filtering & search.
5. **Swap Engine & Escrow Flows**: Dual-pane Swap Interface, Escrow Timeline, Dispute resolution form.
6. **Repair Marketplace**: Technician profile, repair quotes, milestone tracker.
7. **Mobile Screens**: Responsive layouts designed for seamless mobile experiences.

## Technical Context
The designs serve as a foundational reference for frontend developers implementing the Veloxous platform. No code changes are associated with this update; these resources reflect the UI/UX implementation direction.
